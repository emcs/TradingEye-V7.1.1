Member Point Discount
Member Points Member points give the user the opportunity to purchase items from your store and build up points. The points have a currency value set by the administrator, so the more points built up the more credit the customer has available to spend.

Member Point Value Calculation to give a point value. If a customer has 2,000 points, and the point value is set to .03, the customer then has £60.00 worth of points. (2000 X .03 = 60).

Member Point Calculation Calculating value for each unit (£1) purchased. If it is set to 5, then for every £1.00 spent, the customer recieves 5 points (£1.00 = 5 Points).
