Help File: Product Importer [ X ]

The product importer imports product information into Tradingeye. It uses a flat file database (.csv) with various delimiter options.

A flat file is usually setup with one row of data equaling one record and each item in that row is seperated by a delimiter character, Pipe (|), Semi Colon (;), Comma (,).

To use the Tradingeye product importer, you must tell it how your flat file database is setup. There is a series of select lists that are all the same, labeled 1-33, one for each field availble in the database.

The first "column" in your flat file database would corrolate to the first select list, if it is a Product title in your flat file, then select title in the first select list, if description is the second "column" in your flat file, choose description on the number 2 select list and so on.

You will need to select a department for your products to be imported into.

Choose a delimiter

Select a layout and template that you wish to supply to your product(s)

Then select the file from your local hard drive.
