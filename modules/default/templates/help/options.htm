Help File: Standard Options [ X ]

The option editor allows you to create dropdown lists for your product pages. Options like Shirt Size: Small/Medium/Large, Colour: Red/Blue/Green

Each option item can be tracked with stock control, but if you do use stock, you can only have one option per page. If stock is turned on in your Tradingeye admin, the option levels will be adjusted with each order. Once the option item reaches 0, it will be turned off and not available. If you select "Allow Backordering", then the item will NOT be turned off and avaialble for order but with a backorder status.

Options can also have an additional price, if you provide an additional price, it will be calculated by multiplying the quantity * additional price and also an image.
