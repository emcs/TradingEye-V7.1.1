Help File: Feature Settings [ X ]
  • Member Points - Member points give the user the opportunity to purchase items from your store, and build up points. The points have a currency value, the more points built up the more value the customer has.


  • Member Point Value - Calculating value for each unit (£) purchased. If it is set to 2, then for every £1.00 spent, the customer recieves 2 points.


  • Member Point Calculation - Calculation to give a point value. If a customer has 2,000 points, and the point calculation is set to .03, the customer then has £60.00 worth of points. (2000 X .03 = 60).


  • Inventory Tracking - Inventory is an all or nothing proposition. If you choose inventory, it will track inventory for all products, options and choices.


  • Customer Reviews - The default templates that come with Tradingeye Shop have a custom tag that offers reviews for customers to review and give opinions on each product. If you are logged in as an administrator, you can delete offensive reviews.


  • Wish List - Wish lists allow customers to build up a 'wish list' of products that they want and then email friends and family the list. Uncheck to turn off.
