Help File: Custom Options [ X ]

Custom options are very useful tools in Tradingeye. It allows you to have input boxes on your product pages.

You can use the input boxes for a variety of applications.

  • Quantity boxes - If you wish to have multiple choices that can have various quantities.
  • Input Boxes - Input boxes, to add comments, a name or anything else.
  • Textarea Boxes - For large amounts of text that go on a birthday card or something similar.
  • Check boxes - To offer the user the ability to select various extra items.

Each choice can be a stock-controlled item and have additional pricing.

To add a choice to a product, navigate to the product, edit the product and click on the View/Edit Custom Options button/link.

The editor will give you a list of possible choices available, just choose the one you want to add click the >> and it will transfer the choice to that product. In the right hand window, select the choice you want to remove and click the << to remove it from the product.
