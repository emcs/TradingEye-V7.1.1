Manual Bulk Upload

Tab separated text will be generated when you click on this link. All the active products and basic details will be listed in this file.

Use this file to upload products on Google Base.

When on Google Base, There will be three options
1) One at a time
2) Bulk upload
3) API

Select "Bulk upload" option and specify the settings ie.
a) File name to be uploaded.
b) Select the Tab-Separated text file
c) Froogle will process this file and products will be listed on the froogle site.
