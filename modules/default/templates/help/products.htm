Help File: Product Builder [ X ]

The Product Builder is where you set up your products so that they are ready to sell in your online shop.

  • Volume Discount - Click this link if you want to offer volume discounts, and to define these settings.
  • Standard Options - If this product has options and those options have already been created in the "Option Builder" click this link to attach desired options to the product.
  • Custom Options - As with options, click this link to attach custom options to this product.

  • Title - The name or title of the product.
  • Product Code - The unique item number or product code.
  • Price - The price at which you are selling this product.
  • R.R.P. - The recommended retail price at which the item was sold for pricing.
  • Select a Layout - This is the overall look and feel of the site. Usually this is the same for most items in your shop.
  • Select a Template - This is the actual part of the page that diplays this particular product's information.

  • Use Stock control - Check this box to use the stock control feature and track your stock levels.
  • Allow backorders - This allows your stock levels to enter minus amounts.
  • Stock Levels - Set the amount of this particular item you have in stock to begin with.
  • On Order - If an item is backordered and you are using stock controls, you can tell the system how many you have on order.
  • Due Date - If items are "on order", you can put their expected due date.

  • Postage Code - If using postage codes, this will populate with the values set in the postage builder for shipping codes. The default value is "0".
  • Item Weight - The weight of the item, used to calculate postage by weight.
  • Postage Notes - A short line of text to describe the postage details. For example, "Dispatched in 2-3 Days" or "Sent Immediately". This gives your customers more information about the product and it's availability.
  • Free Postage - If checked, this product has no associated postage costs.
  • Supplier - To attach a supplier to your product, you must first have the supplier details set up in your Supplier builder. If you add a logo to the supplier record, the logo will show up on this product's page.

  • Image A, B & C - Three possible images for the product, normally used as (A)Thumbnail, (B)Product Page and (C)Large Detail Image.
  • Downloadable File - Allows a downloadable file to be offered, for example an instructional PDF, sound clip, readme text file, software etc.

  • File Name (URL) - The corresponding URL for this product.
  • Meta Title - The actual title of the page, visible in the browser title bar.
  • Meta Description - Used "behind the scenes" for search engines. This should be a keyword heavy description relative to the product's content and description.
  • Meta Keywords - A list of keywords used to describe the product to search engines, seperated by commas. For example: keyword1, keyword2, keyword3 etc.

  • On / Off - Check this box to have the product turn live on the site.
  • VAT / Tax - Check this box to have the current VAT / Tax settings applied to this product.
  • On Sale - If checked, the system will note that this product has a sale price.
  • Add to Basket Button - If this product is only available for enquiries or not available for sale, Uncheck.
  • Short Description - If using descriptions on the site, this is a brief description of the product.
  • Long Description - If using descriptions on the site, this is a more comprehensive description of the product.
