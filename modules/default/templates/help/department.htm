Help File: Department Builder [ X ]

The Department Builder is where you set up your department structure for your site..

  • Title The name or title of the department or sub department.
  • Select a Layout - This is the overall look and feel of the site. Usually this is the same for all departments and sub departments.
  • Select a Template - This is the actual part of the page that diplays this particular department's information.

  • Image A, B - Two possible images for the department, normally used as (A)Main Navigation level, (B)Sub-Department level
  • File Name (URL) - the corresponding URL for this department.
  • Meta Title - The actual title of the page, visible in the browser title bar.
  • Meta Description Used "behind the scenes" for search engines. This should be a keyword heavy description relative to the department's content and description.
  • Meta Keywords - A list of keywords used to describe the department and it's content to search engines, seperated by commas. For example: keyword1, keyword2, keyword3 etc.

  • On / Off - Check this box to have the department displayed on the site.
  • Short Description - If using descriptions on the site, this is a brief description of the department.
  • Long Description - If using descriptions on the site, this is a more comprehensive description of the department.
