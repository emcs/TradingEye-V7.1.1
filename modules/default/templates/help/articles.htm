Help File: Content Builder [ X ]

  • Title The name or title of your page.
  • Select a Layout - This is the overall look and feel of the site. Usually this is the same for all pages.
  • Select a Template - This is the actual part of the page that diplays this particular content's information.

  • Image A, B & C - Three possible images for the artciles, normally used as (A)Main Article, (B)Custom Image level and (C)Custom Image.
  • File Name (URL) - the corresponding URL for this page.
  • Meta Title - The actual title of the page, visible in the browser title bar.
  • Meta Description Used "behind the scenes" for search engines. This should be a keyword heavy description relative to the articles's content and description.
  • Meta Keywords - A list of keywords used to describe the article and it's content to search engines, seperated by commas. For example: keyword1, keyword2, keyword3 etc.

  • On / Off - Check this box to have the article turn live on the site.
  • Short Description If using descriptions on the site, this is a brief description of the article.
  • Long Description If using descriptions on the site, this is a more comprehensive description of the article.
