Help File: File Manager [ X ]

The File Manager allows you to add, edit and delete files on your webserver.

Main Module

A list of various directories where you can modify/add files.

  • /images/department/ - Site Images are the images uploaded during department product manipulation.
  • /images/product/ - Site Images are the images uploaded during product manipulation.
  • /images/content/ - Site Images are the images uploaded during content manipulation.
  • /images/menu/ - Site Images are the images uploaded during menu manipulation.
  • /images/options/ - Site Images are the images uploaded during option manipulation.
  • /graphics/admin/ - These are images that are uploaded by the user/administration for use within the content and layout of the admin site.
  • /graphics/site/ - These are images that are uploaded by the user/administration for use within the content and layout of the main site.

Add - Gives you the ability to Add/Upload files to the webserver in the specified directory.

View - View files within the specified directory. From the view screen, you and edit or delete files on your webserver.
