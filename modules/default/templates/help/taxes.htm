Help File: VAT/Tax Settings [ X ]

The VAT/Tax builder allows you to specify taxes for various countries and also their Counties/States. Populating the Counties/State for a country also means that the values would be used when users sign up for example and select their Country from the select list.
Tax Code Area (Sage): You only need to set value for this field if you are using Sage plugin

  • T1: UK VAT taxable
  • T4: EU Sales
  • T9: Outside EU
