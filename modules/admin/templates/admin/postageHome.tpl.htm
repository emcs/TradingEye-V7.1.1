Postage & Packing

Default Postage & Packing Method

Choose a base Postage & Packing method, to be applied as your default setting.

Status Postage method Description
Regional Country / State Based Postage
Custom Regional Zones Custom Regional Zones
All Customers Postage that is charged to all customers regardless of location.

Postage Modifiers

Postage options below can work in conjunction with any base Postage & Packing method.

Status Postage method Description
Flat Rate Charge Add a flat charge to this postage method.
Postal Range Postage charge based on range of total order.
Per Item Charge Postage charge based on number of items added to cart.
Weight Based Charge Postage charge based on the total weight of items added to cart.
Zip Code Charge Postage charge based on the zip code of the shipping address.
Postage Group Charge Postage charge that is based on what Postage Group each product is assigned to.
Free Postage Makes postage free if subtotal, and country requirements are met.
Multiple Postage Options Adds postage options. Each postage option has its own charge and is only displayed for specified countries.
