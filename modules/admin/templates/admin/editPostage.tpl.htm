Percentage Rate

If you want to charge a postage rate of 5% of the total order, you would put 5 in the percentage rate box below. You will then need to add a base postage charge. If the 5% is lower than the base postage charge, Tradingeye will charge the base postage charge to the customer.

Postage Ranges

Postage ranges enable you to set price ranges for postage charges. If you set a range from {TPL_VAR_CURRENCY}10 to {TPL_VAR_CURRENCY}20 that has a rate of {TPL_VAR_CURRENCY}14, if you customer spends {TPL_VAR_CURRENCY}30, they would be charged {TPL_VAR_CURRENCY}14 postage. If you want to set your final range to unlimited, then just put '0' or 'unlimited' in the last range field.

Setting special postage options enable you to offer different postage methods to your customers which they can select from a list when checking out. Each option can be limited to only specific countries that are chosen by you. On this page you can create and edit the postage options. If you wish to modify which country shows which postage option, then go here.

Free Postage

Free postage enables you to offer free postage to a specific country once a required subtotal is met. All other countries can be offered a second subtotal allowing them to qualify for free postage or that total can be set to 0 and disable free postage for all other countries.
