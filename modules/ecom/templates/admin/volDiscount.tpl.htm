Volume Discounts

Volume discounts enable you to set a {TPL_VAR_CURRENCY} value discount for products bought in bulk. If you set the Quantity Start value to 1, and the Quantity End value to 10 along with a Discount Value of {TPL_VAR_CURRENCY}10, then should a customer purchase 8 products, they will be given a {TPL_VAR_CURRENCY}10 discount on their order. If you would discount any order over a certain amount, then leave a 0 in the Quantity End field.
